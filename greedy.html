Greedy Algorithms

Make locally optimal choices to find global solutions

What is a greedy algorithm?

A greedy algorithm is an algorithm that makes an optimal local choice in order to find the globally optimal solution.

Definitions

  • Global Optimality: The best possible solution for the entire problem.
  • Local Optimality: The best possible choice at each individual step of the algorithm.

Steps for greedy:

  1. Start With a Small Example
  2. Test Different Local Choices
  3. Try Writing It in Code, Analyze Space and Time Complexity
