Recursion

Break down complex problems into smaller, similar subproblems

What is Recursion?

Recursion is when a function solves a problem by calling itself on smaller subproblems. It's like solving a puzzle by breaking it into smaller pieces that follow the same pattern.

Definitions

  • Base Case: The stopping condition where the recursion ends. This is crucial to prevent infinite recursion.
  • Recursive Call: Where the function calls itself with a smaller or simpler version of the problem.

Steps for Recursion:

  1. Start With a Small Example
  2. Turn the Idea Into a Repeatable Action (Recursive Call)
  3. Stop When You Hit the Smallest Case (Base Case)
  4. Try Writing It in Code, Analyze Space and Time Complexity
  5. Optimize
